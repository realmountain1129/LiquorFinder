# LiquorFinder

You can find various liquors by taking pictures.

Here's the list of liquors:
* Gordon's Gin
* Absolute Vodka
* Bombay Sapphire
* Smirnoff
* Barcardi Carta Blanca
* Jose Cuervo
* Jack Daniel's
* Tanqueray
* Johnnie Walker


## Built with

* Python3
* Tensorflow
* Swift
* CoreML
* Vision
* UIKit
